# Calculadora de Gorjeta

Este projeto é uma calculadora de gorjetas que permite ao usuário inserir o valor de uma conta e selecionar uma porcentagem de gorjeta para calcular o valor da gorjeta e o total a pagar. A interação é feita diretamente na página por meio de JavaScript, e o layout é construído com HTML e CSS.

O objetivo deste projeto é praticar a manipulação de entrada de usuário, cálculo de valores e exibição dinâmica de resultados utilizando JavaScript, junto com a estruturação e estilização de páginas web.

## Tecnologias Utilizadas

- HTML5
- CSS3
- JavaScript

## Funcionalidades

- Entrada do valor da conta
- Escolha de porcentagem de gorjeta
- Cálculo automático de valor de gorjeta
- Exibição do valor total atualizado
- Interface simples e funcional
